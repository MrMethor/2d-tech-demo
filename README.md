## Description
Top down 2D tile-scrolling tech demo made with [SFML](https://www.sfml-dev.org/).
- Status - DISCONTINUED

### Features
* Basic 2D movement
* Tile scrolling camera centered on player
* FPS and UPS differentiation with interpolation
* Basic zoom function

## Running
* Windows 10
* Download the [portable version](https://github.com/MrMethor/game/releases/tag/game) from the release page.
* To run the game open extract the zip file and run the Game.exe
### Controls
- WASD movement
- Scroll-wheel to zoom in and out
- F for fullscreen

## Building
* The project was built using [CLion](https://www.jetbrains.com/clion/) and [SFML 2.6.2](https://www.sfml-dev.org/download/sfml/2.6.2/)
* The code includes a CMake file to ease the setup
